// Camadas do framework
D
Directive
O Cérebro Estratégico → "O quê"
📋

Arquivos Markdown (.md) que funcionam como SOPs — Procedimentos Operacionais Padrão. Definem objetivos, entradas, passos e critérios de sucesso em linguagem natural. É o "playbook" da empresa: a IA lê isso primeiro antes de qualquer coisa.

agents.md SOPs em linguagem natural Critérios de sucesso /directives folder PRD como input
O
Orchestration
O Gerente de Projeto → "Quando e como"
🧠

O modelo de IA (Claude, Gemini, GPT) atuando como gerente: decide a ordem das tarefas, qual ferramenta usar, coordena o fluxo baseado nas diretivas. Responsável por roteamento e julgamento — não pelo trabalho pesado. Pode ser trocado sem reconstruir a infraestrutura.

Roteamento inteligente Chama ferramentas MCP Loop PTMRO Independente do modelo Coordena subagentes
E
Execution
As Mãos Determinísticas → "Como, de verdade"
⚙️

Scripts Python atômicos que realizam o trabalho pesado de forma repetível e sem alucinações. Chamadas de API, processamento de dados, envio de e-mails — tudo que precisa ser determinístico fica aqui. A IA decide o que fazer; o Python faz de fato.

// Heurísticas de decisão
Regra 10x
Só otimize se houver caminho claro para tornar o fluxo 10x melhor em tempo, custo ou precisão.
Human-in-the-Loop
Alta magnitude financeira ou alta sensibilidade de qualidade → sempre inclua revisão humana.
Alavancagem Horizontal
Automatizar 90% de 10.000 micro-tarefas vale mais do que 100% de um único cargo complexo.
Limite de Paralelização
Ponto ideal: 2–3 agentes simultâneos. Acima de 4, o ganho é anulado pela sobrecarga de atenção.
Densidade de Prompt
Remova saudações e palavras inúteis. Prompts mais curtos e densos geram saídas melhores.
Vibe Coding
Se você está editando código manualmente, está errado. Aja como CEO: dê ordens, a IA executa.
// Erros comuns e como evitá-los
Poluição de Contexto
Use subagentes em janelas isoladas. Compacte históricos longos.
Execução Probabilística
IA decide (orquestração), Python executa (determinístico). Nunca misture.
Segredos Expostos
Sempre use .env para chaves de API. Instrua a IA a nunca alterar esse arquivo.
Token Burn
Carregue apenas servidores MCP estritamente necessários para a tarefa.
Falta de PRD
Erros no início se multiplicam exponencialmente. Planejamento humano é obrigatório.
